How to fix 'Fatal error: Allowed memory size of xxxx bytes exhausted'

After working with WordPress for several years, the "Allowed memory size exhausted" error is one of the more common errors that you will run into. So how do I fix it? First and easiest thing to try is to increase the PHP memory limit by adding this line of code to the wp-config.php file like so:
define( 'WP_MEMORY_LIMIT', '256M' );
If this still does not work, try the following: Go to home/username/public_html/wp-includes/default-constants.php Open the file and replace the following:
// Define memory limits.
if ( ! defined( 'WP_MEMORY_LIMIT' ) ) {
if ( false === wp_is_ini_value_changeable( 'memory_limit' ) ) {
define( 'WP_MEMORY_LIMIT', $current_limit );
} elseif ( is_multisite() ) {
define( 'WP_MEMORY_LIMIT', '64M' );
} else {
define( 'WP_MEMORY_LIMIT', '40M' );
} }
with the following code:
